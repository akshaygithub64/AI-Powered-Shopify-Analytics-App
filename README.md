# AI-Powered Shopify Analytics App

This is an AI-powered analytics application for Shopify data.

## Features

- Analyze customer data
- Inventory management
- Order processing
- AI insights

## Installation

1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `python app.py`

## Data

The `data/` folder contains sample CSV files:

- `customers.csv`: Customer information
- `inventory.csv`: Inventory details
- `orders.csv`: Order records
- `products.csv`: Product catalog

## Services

- `ai_service.py`: AI-related functionalities
- `shopify_service.py`: Shopify integration
